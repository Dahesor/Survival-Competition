# 进度争霸赛

### 坐稳扶好……试运行压测版本准备进度99.99%!

## 简述

这是一个进度竞速 + 饥饿游戏结合的缝合怪。游戏分为三个阶段，时长约为90~105分钟。游戏开始时，绝大部分合成台配方均被锁定。玩家完成进度可以为自己小队的所有成员解锁部分合成配方，并获得分数。
 - 一阶段（进度竞速）：所有玩家在世界中心出生。世界边界将在接下来的10分钟内从48格扩张到1024。下界与末地开放。玩家需要尽可能低抢险完成进度以获取分数，并解锁有用的合成配方。
 - 二阶段（山丘之王）：在完成进度的基础上，地图中会出现占领点。占领占领点的队伍持续获得分数。占领点每隔5分钟将会移动到新的随机位置。
 - 三阶段（生死逃杀）：末地关闭，主世界与下界的世界边界开始收缩；此阶段团灭的小队将被淘汰。小队存活的名词越高，计算最终得分时就会使用更高的倍率。

## 安装说明

游戏当前使用1.21.11版本。不可以使用其他版本。

### 客户端

所有玩家与旁观者都需要安装专属资源包，否则无法正常进行游戏。\
兼容的客户端：
- 原版1.21.11客户端
- Fabric 1.21.11最新版客户端
  - 确保可以正常兼容的模组：
    - Sodium
    - Command Keys

### 服务端

可确保正确运行的服务端（受维护）：
- 原版1.21.11服务端
- Fabric 1.21.11 最新版服务端
  - 确保可以正常兼容的模组：
    - Lithium

一定**不要**使用（已知会出现问题）：
- Paper或Spigot系服务端
- Forge或NeoForge系服务端

未提到的服务端没有已知的问题，但也不受保证或维护。

#### 服务端准备

修改[server.properties](https://zh.minecraft.wiki/w/%E6%9C%8D%E5%8A%A1%E7%AB%AF%E9%85%8D%E7%BD%AE%E6%96%87%E4%BB%B6%E6%A0%BC%E5%BC%8F)文件（单人或局域网请跳过）:
```
function-permission-level=2
gamemode=adventure
generate-structures=true
spawn-protection=0
```

简单地开启一张原版生存地图，并安装提供的数据包。由于数据包更改了世界生成规则，你必须在**世界第一次加载的时候就已经安装数据包**。

如果你不知道怎么做：\
**专用服务器**
 - 确保服务器已关闭
 - 找到服务器的目标世界文件夹（默认为world）
 - 若其中有内容，全部删除（如果需要记得备份）
 - 在文件夹内创建datapacks文件夹，将数据包放置于datapacks文件夹里
 - 启动服务器
**单机/局域网**\
 - 进入创建新世界的页面
 - 打开“更多”选项卡
 - 打开“数据包”选项
 - 导入数据包，并置于加载列表最上方
 - 创建世界

#### 地图加载

启动服务器后，**先不要让玩家加入**。应该由一名拥有管理员权限的玩家进入服务器，并配合系统完成地图初始化功能，这大概需要三分钟。

当游戏内提示准备完成后，就可以让其他玩家进入了。
